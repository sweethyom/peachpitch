## 문제가 있었던 것
-  프로젝트 규모에 대한 고민
- git lab master branch 소멸
- npm으로 react 실행 과정에서의 시행 착오
    - 앱의 package.json "react": "^18.0.0", "react-dom": "^18.0.0" 버전 수정
    - npm install web-vitals 설치
    - npm start 하면 해결
- vite + react로 해결
    - npm create vite@latest [[my-react-app] -- --template react]
    - [my-react-app]으로 이동
    - npm install
    - npm run dev 하면 해결결
